# Pandas Practice Projects

This repository contains structured pandas projects created to practice data analysis, data cleaning, and real-world dataset exploration.

## Project Structure

### 01_pandas_basics
- Loading CSV data
- Basic exploration (`head`, `info`, `describe`)
- Filtering rows and selecting columns
- Saving processed outputs

### 02_data_cleaning
- Handling missing values
- Renaming columns
- Fixing data types
- Removing duplicates

### 03_real_world_analysis
- Real-world dataset analysis (DDoS traffic)
- Feature selection
- Filtering suspicious traffic
- Dataset excluded due to GitHub size limits

### outputs
- Saved CSV results generated from analysis scripts

## Tools & Libraries
- Python
- Pandas

## Notes
Large datasets are excluded from version control. Instructions to download them are provided inside the respective project folders.
